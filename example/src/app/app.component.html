Angular Color Picker Directive

A Color Picker Directive for Angular with no dependencies.

by Alberto Pujante

View on Github

Usage:

<input [(colorPicker)]="color"
       [style.background]="color"/>

Or:

<input [colorPicker]="color"
        (colorPickerChange)="color=$event"
        [style.background]="color"/>

Show the color in the input field:

<input [(colorPicker)]="color"
       [style.background]="color"
       [value]="color"/>


Output format:

<input [(colorPicker)]="color"
       [style.background]="color"
       [value]="color"
       [cpOutputFormat]="'rgba'"/>

Changing dialog position:

<input [(colorPicker)]="color"
       [style.background]="color"
       [value]="color"
       [cpPosition]="'bottom'"/>
Change me!

You can introduce a offset of the color picker relative to the html element:

<span [(colorPicker)]="color"
      [cpPosition]="'bottom'"
      [style.color]="color"
      [cpPositionOffset]="'50%'"
      [cpPositionRelativeToArrow]="true">Change me!</span>

Show cancel button:

<input [(colorPicker)]="color"
       [style.background]="color"
       [value]="color"
       [cpCancelButton]="true"/>

Change cancel button class, in this example we are using a bootstrap button:

<input [(colorPicker)]="color"
       [style.background]="color"
       [value]="color"
       [cpCancelButton]="true"
       [cpCancelButtonClass]= "'btn btn-primary btn-xs'"/>

Show OK button:

<input [(colorPicker)]="color"
       [style.background]="color"
       [value]="color"
       [cpOKButton]="true"
       [cpSaveClickOutside]="false"
       [cpOKButtonClass]= "'btn btn-primary btn-xs'"/>

With preset colors:

<input [(colorPicker)]="color"
       [style.background]="color"
       [cpPresetColors]="['#fff', '#000', '#2889e9', '#e920e9', '#fff500', 'rgb(236,64,64)']"/>


Toggle status: {{toggle2}}

Use cpToggle with cpIgnoredElements:

<input #ignoredInput [(colorPicker)]="color"
       [(cpToggle)]="toggle"
       [style.background]="color"
       [cpIgnoredElements]="[ignoredButton, ignoredInput]"/>

<button (click)="toggle=!toggle;lastColor=color"
         #ignoredButton></button>


Change alpha channel behaviour:

<input [(colorPicker)]="color5"
       [style.background]="color"
       [cpOutputFormat]="'rgba'"
       [cpAlphaChannel]="'disabled'"
       [value]="color"/>

<input [(colorPicker)]="color4"
       [style.background]="color"
       [cpOutputFormat]="'rgba'"
       [cpAlphaChannel]="'always'"
       [value]="color"/>

<input [colorPicker]="color"
       (colorPickerChange)="rgbaText=onChangeColorHex8($event);color=$event"
       [style.background]="rgbaText"
       [cpOutputFormat]="'hex'"
       [cpAlphaChannel]="'always'"
       [value]="color"/>

Show the dialog permanently: 

<span [(colorPicker)]="arrayColors[selectedColor]"
      [cpDialogDisplay]="'inline'"
      [style.background]="arrayColors[selectedColor]"
      [cpCancelButton]="true"
      [cpToggle]="true"></span>

<div [style.background]="arrayColors['color']"
     (click)="selectedColor='color'"></div>

<div [style.background]="arrayColors['color2']"
     (click)="selectedColor='color2'"></div>

...

Add and remove preset colors:

<input [(colorPicker)]="color"
       [style.background]="color"
       [cpPresetColors]="['#fff', '#2889e9']"
       [cpOutputFormat]="'rgba'"
       [cpAlphaChannel]="'always'"
       [cpAddColorButton]="true"/>

Options Values (default values in bold)
cpOutputFormat 'hex', 'rgba', 'hsla'
cpPosition 'right', 'left', 'top', 'bottom'
cpPositionOffset '0%'
Dialog offset (percent) relative to the element that contains the directive.
cpPositionRelativeToArrow false, true
Dialog position is calculated relative to the dialog (false) or relative to the dialog arrow (true).
cpWidth '230px'
Use this option to set color picker dialog width (pixels).
cpHeight 'auto'
Use this option to force color picker dialog height (pixels).
cpSaveClickOutside true, false
If true the initial color is restored when user clicks outside.
cpOKButton false, true
Shows the Ok button. Saves the selected color.
cpOKButtonText 'OK'
cpOKButtonClass Class to customize the OK button.
cpCancelButton false, true
Shows the Cancel button. Cancel the selected color.
cpCancelButtonText 'Cancel'
cpCancelButtonClass Class to customize the Cancel button.
cpFallbackColor '#fff'
Is used when the color is not well-formed or not defined.
cpPresetLabel 'Preset colors'
Label for preset colors if any provided used.
cpPresetColors []
Array of preset colors to show in the color picker dialog.
cpToggle false, true
Input/ouput to open/close the color picker.
cpIgnoredElements []
Array of HTML elements that will be ignored by the color picker when they are clicked.
cpDialogDisplay 'popup', 'inline'
popup: dialog is showed when user clicks in the directive.
inline: dialog is showed permanently. You can show/hide the dialog with cpToggle.
cpAlphaChannel 'enabled', 'disabled', 'always'
enabled: alpha channel is not included for hexadecimal values or for values without alpha (alpha = 1).
always: alpha channel is included for hexadecimal values and values without alpha (alpha = 1).
disabled: alpha channel is completely disabled.
cpUseRootViewContainer false, true
Create dialog component in the root view container instead the elements view container.
cpAddColorButton false, true
Add or remove colors into your preset panel. The [cpPresetColors] is needed
cpAddColorButtonText 'Add color'
cpAddColorButtonClass Class to customize the add color button.
cpRemoveColorButtonClass Class to customize the remove color button.
cpMaxPresetColorsLength 8 (number)
Use this option to set the max colors allowed into preset panel.
cpPresetEmptyMessage 'No colors added'
Message for empty colors if any provided used.
cpPresetEmptyMessageClass Class to customize the empty colors message.

Events Description (data format in bold)
colorPickerChange Changed color value, send when color is changed. (value: string)
colorPickerSelect Selected color value, send when user presses the OK button. (value: string)
cpToggleChange Status of the dialog, send when dialog is opened / closed. (open: boolean)
cpInputChange Input name and its value, send when user changes color through inputs. ({{ '{' }}input: string, value: string{{ '}' }})
cpSliderChange Slider name and its value, send when user changes color through slider. ({{ '{' }}slider: string, value: Object{{ '}' }})
cpPresetColorsChange Preset colors value, send when Add Color button is pressed. (value: array)
